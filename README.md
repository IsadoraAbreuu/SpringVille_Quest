# 🌆 SpringVille Quest

SpringVille Quest é um jogo educativo e interativo inspirado em cidades animadas, onde o jogador tem a missão de restaurar os principais pontos da cidade completando desafios, quizzes e atividades lúdicas.

Ao concluir cada missão, o jogador ganha moedas, desbloqueia novas áreas e devolve as cores para SpringVille até restaurar completamente a cidade.

---

## 🎮 Sobre o Jogo

A cidade de SpringVille perdeu suas cores e sua energia.

Para restaurá-la, o jogador deverá visitar diferentes locais da cidade, resolver desafios e completar missões especiais.

Cada local possui atividades únicas, como:

- ❓ Quizzes
- 🧩 Ordenação de etapas
- 🎯 Desafios interativos
- 🏆 Sistema de recompensas
- 🪙 Coleta de moedas

Ao concluir todas as localidades, o jogador recebe uma celebração final e o título de:

**Herói de SpringVille**

---

## 🗺️ Locais Disponíveis

### 🌳 Parque
Primeira missão para iniciar a restauração da cidade.

### 🏛️ Prefeitura
Tomada de decisões importantes para SpringVille.

### 🏫 Escola
Desafios de conhecimento e aprendizado.

### 🏠 Casa
Missões relacionadas à organização e rotina.

### 🛒 Mercado
Última etapa antes da restauração completa.

---

## 🪙 Sistema de Recompensas

Cada missão concluída concede moedas ao jogador.

As moedas representam o progresso da restauração da cidade e são acumuladas durante toda a jornada.

## 🚀 Como Executar

### 1. Clonar o projeto

```bash
git clone https://github.com/seu-usuario/SpringVille_Quest.git
```

### 2. Entrar na pasta

```bash
cd app
```

### 3. Instalar dependências

```bash
npm install
```

### 4. Executar

```bash
npm start
```

O projeto ficará disponível em:

```bash
http://localhost:3000
```
